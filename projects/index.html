Badge Giver

Badge Giver is a simple tool. It is designed to help Discord users obtain the Active Developer Badge without the need for coding or technical knowledge. The project uses JavaScript and aims to provide an easy and accessible way for users to acquire the Active Developer Badge within the Discord platform. Here are some of the key features Badge Giver offers:

  • Easy Badge Acquisition: Obtain the Active Developer Badge on Discord effortlessly without any coding or technical skills required.
  • No Coding Required: Eliminates the need for users to write code to obtain the badge.
  • Free of Charge: Access the Active Developer Badge for free without any hidden costs or subscriptions.

Zubb

Zubb is a bot designed to enhance your Discord server experience but it is since been discontinued. Here are some of the key features Zubb offers:

1. Moderation Tools:

  • Auto-moderation: Automatically detect and manage inappropriate behavior with customizable filters for spam, offensive language, and more.
  • Manual Moderation: Commands to mute, kick, ban, and warn users directly from the chat.

2. Utility Commands:

  • User Info: Look up information about specific users, such as join date and role assignments.
  • Server Info: Retrieve detailed information about your server, including member count, roles, and channels.

3. Economy System:

  • Virtual Currency: Earn virtual currency within your server by claiming dailies.
  • Level System: Earn levels within your server by chatting.
  • Leaderboards: Compete with other members to see who can amass the most wealth.

4. [Ideas] Fun Commands:

  • Rock Paper Scissors: Play Rock Paper Scissors with a bot in your Discord Server!
